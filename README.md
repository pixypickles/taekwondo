# Taekwondo Mobile Prototype

Godot 4.x 向けの、スマホ用テコンドー対戦ゲームの最小プロトタイプです。

## 現在入っているもの

- 青プレイヤー / 赤CPUの簡単キャラクター
- 白道着ではなく、青・赤のわかりやすい服
- 顔に目あり
- 左右移動
- 前前：ステップイン
- 後後：バックステップ
- ↓：回転フェイント
- Z：小キック
- X：大キック
- C：ガード
- 回転中 + Z：ティッチャギ（後ろ蹴り）
- 回転中 + X：ティフリギ（後ろ回し蹴り）
- 上 + Z：ネリョチャギ
- 上 + X：跳び後ろ回し蹴り
- 体力ゲージ
- バランスゲージ

## PCテスト用キー

| 操作 | キー |
|---|---|
| 移動 | ← / → |
| ステップイン | 前方向を素早く2回 |
| バックステップ | 後方向を素早く2回 |
| 回転フェイント | ↓ |
| 小キック | Z |
| 大キック | X |
| ガード | C |
| ネリョチャギ | ↑ + Z |
| 跳び後ろ回し蹴り | ↑ + X |
| ティッチャギ（後ろ蹴り） | ↓で回転中 + Z |
| ティフリギ（後ろ回し蹴り） | ↓で回転中 + X |

## Godotで開く方法

1. Godot 4.x を起動
2. Import を選択
3. このフォルダ内の `project.godot` を選択
4. Import & Edit
5. 再生ボタンで実行
