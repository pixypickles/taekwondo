# Taekwondo Prototype Source v001

ブラウザで `index.html` を開くと動きます。

## 操作

- ← / →: 移動
- ←← / →→: ステップ
- 回: 回転フェイント
- 回転中 + 小: ティッチャギ
- 回転中 + 大: ティフリギ
- 小: アプチャギ
- 大: トルリョチャギ
- 上 + 小: ネリョチャギ
- 上 + 大: 跳びティフリギ
- 後 + 小: 下がりトルリョチャギ
- 後 + 大: 下がり跳びティッチャギ

## 今回の狙い

ZIPを毎回作り直す方式から、ソースコードを育てる方式に切り替えるための土台です。
技の見た目は `js/moves.js` と `js/renderer.js` を中心に調整します。
