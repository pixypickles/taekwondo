# Taekwondo Duel v1

GitHub Pages用の単一ページHTML5ゲームです。

## 反映した動き
- 下：回転フェイント
- 回転中＋小：ティッチャギ（後ろ蹴り）
- 回転中＋大：ティフリギ（後ろ回し蹴り）
- 上＋小：ネリョチャギ
- 上＋大：跳び後ろ回し蹴り
- 後＋小：下がりトルリョチャギ
- 後＋大：下がり跳びティッチャギ
- ガード、HP、バランスゲージ、CPU入り

## 置き方
このフォルダの中身を GitHub の taekwondo リポジトリ直下に置いてください。
`index.html` がトップにある状態なら GitHub Pages で開けます。
