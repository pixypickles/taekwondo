# Taekwondo Mobile Prototype

GitHub Pagesでそのまま遊べるHTML5版です。

## 操作

- ← / →：移動
- 前方向を素早く2回：ステップイン
- 後方向を素早く2回：バックステップ
- ↓：回転フェイント
- Z / 小：小キック
- X / 大：大キック
- C / ガード：ガード
- 回転中 + Z：ティッチャギ（後ろ蹴り）
- 回転中 + X：ティフリギ（後ろ回し蹴り）
- ↑ + Z：ネリョチャギ
- ↑ + X：跳び後ろ回し蹴り

## GitHub Pagesに置くファイル

リポジトリ直下に以下を置いてください。
