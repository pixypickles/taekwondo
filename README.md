# Taekwondo Mobile Prototype

GitHub Pagesで動くHTML5版テコンドー対戦ゲームのプロトタイプです。

## 操作

| 操作 | キー |
|---|---|
| 移動 | ← / → |
| ステップイン | 前方向を素早く2回 |
| バックステップ | 後方向を素早く2回 |
| 回転フェイント | ↓ |
| 小キック | Z / 画面の「小」 |
| 大キック | X / 画面の「大」 |
| ガード | C / 画面の「ガード」 |
| ネリョチャギ | ↑ + Z |
| 跳び後ろ回し蹴り | ↑ + X |
| ティッチャギ（後ろ蹴り） | 回転中 + Z |
| ティフリギ（後ろ回し蹴り） | 回転中 + X |

## GitHub Pages

このフォルダの中身をリポジトリ直下に置くと、`index.html` がトップページとして表示されます。
