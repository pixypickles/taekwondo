# Game Design Notes

## コンセプト

蹴り・距離・回転フェイントを中心にした、スマホ向け2Dテコンドー対戦ゲーム。
パンチやしゃがみ攻撃ではなく、ステップと蹴りの読み合いを中心にする。

## 重要な方針

- 下入力はしゃがみではなく回転フェイント専用
- 回転中に小キックでティッチャギ（後ろ蹴り）
- 回転中に大キックでティフリギ（後ろ回し蹴り）
- ガードボタンあり
- 体力とは別にバランスゲージあり

## ボタン構成

スマホ想定：

- 方向キー
- 小キック
- 大キック
- ガード

## 技リスト

| 入力 | 技 |
|---|---|
| 小 | 小キック |
| 大 | 大キック |
| 前 + 小 | 前進小キック |
| 前 + 大 | 前進大キック |
| 後 + 小 | 下がりトルリョチャギ |
| 後 + 大 | 下がり跳びティッチャギ |
| 下 | 回転フェイント |
| 回転中 + 小 | ティッチャギ（後ろ蹴り） |
| 回転中 + 大 | ティフリギ（後ろ回し蹴り） |
| 上 + 小 | ネリョチャギ |
| 上 + 大 | 跳び後ろ回し蹴り |
